TimoVM's Nickname Writer code converter

Intended to be used with TimoVM's Nickname Writer setup as outlined here. NicknameConverter based on the original implementation by Scotteh.

Please note that all blank tiles at the end of a nickname need to be entered one by one manually. This will prevent the Nickname Writer from producing an incorrect result.

Underneath each nickname you can find a copy of the nickname, represented in flat text. Please mind the following special characters:

  • [SPACE] represents a blank space
  • * represents a multiplication marker
  • Pk represents the Pk symbol
  • Pk represents the Mn symbol
